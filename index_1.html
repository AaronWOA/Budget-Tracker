First-time setup

To sync between you and your partner in real-time, you'll need a free Firebase project. Takes about 3 minutes.

  1. Go to console.firebase.google.com and click Add project. Name it anything (e.g. "our-budget"), skip Google Analytics.
  2. In your project, click the Web icon (</>) to register an app. Give it any nickname. Skip hosting.
  3. Copy the firebaseConfig object it shows you and paste it below.
  4. In the left sidebar, click Build → Realtime Database → Create Database. Choose any location, then select Start in test mode.
  5. (Optional) Send your partner this same web page link — they'll paste the same config and you'll both see the same data.

Use this option if you just want to track on this one device — no syncing with anyone else. All data stays in your browser.

Your Firebase config is saved only in your browser's storage — it's not sent anywhere except to your own Firebase project.
